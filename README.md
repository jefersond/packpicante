# Pack R3D — Landing Page

Landing page do Pack R3D. Deploy na Vercel; domínio: packpicante.online.

## Como fazer alterações

1. Edite os arquivos (ex.: `index.html`, textos, imagens em `assets/`).
2. Salve e faça commit e push para o GitHub:
   ```bash
   git add .
   git commit -m "Descrição da alteração"
   git push origin main
   ```
3. A Vercel atualiza o site automaticamente em alguns segundos.

## Rodar localmente

```bash
npm run dev
```

Abre em http://localhost:3000 (ou a porta que aparecer).
